Experience WinDesk Cloud before committing to a plan. Three demo environments are available immediately in your browser — no account, no installation, and no sign-up required.

<Note>
  These are shared public demo environments. They may occasionally be unavailable or slow during periods of high traffic. For a dedicated experience with full performance, start a free trial.
</Note>

## Demo environments

<Tabs>

  <Tab title="Ubuntu Desktop">
    **Ubuntu Desktop 22.04** — Full Linux environment with development tools via noVNC.

    The Ubuntu demo gives you a complete Ubuntu 22.04 graphical desktop running in your browser via noVNC. It is pre-configured with a standard set of developer tools, making it ideal for developers evaluating WinDesk Cloud as a cloud Linux workstation.

    **What you can try:**
    - Full GNOME desktop environment
    - Terminal with bash, curl, git, and standard utilities
    - File Manager for exploring the filesystem
    - Text editor and basic development tools

    **Available on:** Business and Enterprise plans
  </Tab>

  <Tab title="Windows Desktop">
    **Windows Desktop** — Full Windows environment via TSplus HTML5 Remote Desktop.

    The Windows demo connects to a live Windows environment served via TSplus HTML5 Remote Desktop. You get a complete Windows desktop in your browser, including the Start menu, taskbar, and file system.

    **What you can try:**
    - Windows 11 desktop environment and shell
    - Office-compatible applications
    - File Manager and system settings
    - Full RDP-grade experience via browser

    **Access:** [win11.blueedge.me](https://win11.blueedge.me/)

    **Available on:** Business and Enterprise plans
  </Tab>

  <Tab title="VS Code Online">
    **VS Code Online** — Browser-based development environment, available on all plans.

    The VS Code demo opens [github.dev](https://github.dev/microsoft/vscode) — a fully functional VS Code instance running in your browser. This demonstrates the browser-based coding workflow you can use on any WinDesk Cloud desktop.

    **What you can try:**
    - Full VS Code editor interface
    - Extensions from the VS Code Marketplace
    - Git integration and file explorer
    - Integrated terminal (within github.dev limitations)

    **Note:** The github.dev instance is hosted by GitHub, not by NeoSC. On a paid WinDesk Cloud desktop, VS Code runs natively on your virtual machine with full terminal and filesystem access.

    **Available on:** All plans
  </Tab>

</Tabs>
